SASE Annual Banquet 2025–2026
A Django web application for the Society of Asian Scientists and Engineers (SASE) Annual Banquet 2026 at Friendship Gardens — Student Pavilion, Senita Ballroom BC.

Event Details
Date: April 18th, 2026
Time: 5:00 – 8:00 PM
Location: Student Pavilion · Senita Ballroom BC
Dress Code: Semi-formal
RSVP Required
Features
Animated envelope invitation card with 3D tilt and holographic foil effect
Falling cherry blossom petals
Parallax scrolling invited section
Live countdown timer to the banquet
April 2026 event calendar
Ticket payment button (Venmo)
RSVP form integration
Tech Stack
Python / Django 5.2
Vanilla HTML, CSS, JavaScript
Google Fonts (Great Vibes, Cormorant Garamond, Cinzel)
Setup
# Install dependencies
pip install -r requirements.txt

# Run development server
python3 manage.py runserver
